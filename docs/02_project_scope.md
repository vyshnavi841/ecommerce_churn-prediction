# Project Scope

## In Scope
- **Customer-level churn prediction**: Developing a predictive model based on historical transaction data.
- **RFM Analysis**: Calculating Recency, Frequency, and Monetary values to segment customers.
- **Feature Engineering**: Creating behavioral patterns and product affinity metrics to aid prediction.
- **Model Comparison**: Evaluating multiple machine learning classification models.
- **Web App Deployment**: Simple application interface deployed for stakeholders to use predictions.

## Out of Scope
- **Real-time predictions**: Predictions will be batch-processed or on-demand via the app, not strictly real-time streaming.
- **Product recommendations**: Recommender systems are excluded from this churn prediction focus.
- **Inventory optimization**: Supply chain analytics are not part of this project.

## Timeline
- **Duration**: 8 weeks.
- Covers data collection, cleaning, EDA, feature engineering, modeling, evaluation, and deployment.

## Constraints
- **Cost**: Free tools only, no paid APIs.
- **Deployment**: Must use a public deployment platform (Streamlit Cloud).
- **Data**: Must use the provided UCI Online Retail dataset.
